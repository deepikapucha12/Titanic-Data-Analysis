# Titanic Data Analysis

This project analyzes the Titanic dataset to understand survival patterns using Python.

## Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL (example queries)

## What I Did:
- Cleaned missing data in Age and Embarked columns
- Analyzed survival patterns using Pandas DataFrames
- Visualized data using Matplotlib (bar charts for survival count, survival by gender, survival by class)
- Used Seaborn heatmap to explore correlations between features
- Wrote example SQL queries for survival analysis

## Key Insights:
- Females had higher survival rate than males
- First class passengers survived more than lower classes
- Survival is affected by multiple factors such as gender, class, and age
